Using

在HTML上添加tooltips.min.css

<link rel="stylesheet" href="tooltips.min.css">

在 stylus 中使用

@import "node_modules/tooltips"

给标签添加 data-tooltips 和 class="tooltips-top" 属性

<span data-tooltips="top" class="tooltips-top"" >tooltips</span>
