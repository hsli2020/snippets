The flex container properties:

  • flex-direction子元素排列方向(row,column)
  • flex-wrap
  • flex-flow (flex-direction, flex-wrap)
  • justify-content子元素在主轴上排列方式
  • align-items子元素在交叉轴上的排列方式
  • align-content

The flex item properties are:

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex (flex-grow, flex-shrink, flex-basis)
  • align-self

Perfect Centering

.flex-container {
  display: flex;
  height: 300px;
  justify-content: center;
  align-items: center;
}
